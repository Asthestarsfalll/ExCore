

## load

```python
def load(filename: str, dump_config: Optional[str] = None, base_key: str = BASE_CONFIG_KEY) -> LazyConfig
```

Load target toml file. `dump_config` is used for dump all loaded config into a single file. `base_key` is used for config inheritance, default to `__base__`.

## build_all

```python
def build_all(cfg: LazyConfig) -> Tuple[ModuleWraper, dict]
```

Build all modules. First element is primary modules, second is other information.


## silent

```python
def silent()
```
